💡 QUICK REFERENCE
═══════════════════════════════════════════════════════════════════════════════

API Endpoints:
• GET    /api/wishlist              - Get all items
• POST   /api/wishlist/add          - Add product
• DELETE /api/wishlist/:id          - Remove product
• GET    /api/wishlist/count        - Get count
• GET    /api/wishlist/check/:id    - Check if exists
• DELETE /api/wishlist              - Clear all

JavaScript Functions:
• getWishlist()
• addToWishlist(productId)
• removeFromWishlist(itemId)
• checkProductInWishlist(productId)
• updateWishlistButtons()
• updateWishlistCount()
• showMessage(msg, type)

HTML Element Attributes:
• data-action="wishlist"       - Mark as wishlist button
• data-product-id="ID"         - Product ID
• data-wishlist-id="ID"        - Wishlist item ID
• class="wishlist-count"       - Count badge


⚡ QUICK COMMANDS
═══════════════════════════════════════════════════════════════════════════════

Database:
  1. Open MySQL/phpMyAdmin
  2. Select database: computer_store
  3. Paste and run: wishlist_schema.sql

Backend:
  1. Copy 3 files to backend/ folder
  2. server.js already updated
  3. Restart: npm start

Frontend:
  1. Copy wishlist.js to frontend/assets/js/
  2. Copy wishlist.html to frontend/pages/
  3. Add <script src="../assets/js/wishlist.js"></script> to pages
  4. Add data-action="wishlist" to buttons
